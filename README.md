# Spring Boot Bank App (Dockerized)

## 🚀 Project Overview
This project is a Spring Boot application containerized using Docker and Docker Compose with MySQL database integration.

## 🧱 Tech Stack
- Java (Spring Boot)
- MySQL
- Docker
- Docker Compose

## ⚙️ Features
- User Registration & Login
- MySQL Database Integration
- Multi-container setup using Docker Compose

## 🐳 How to Run

```bash
docker compose up --build
